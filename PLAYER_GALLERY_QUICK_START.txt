===============================================================================
  STEP 1: TAG PLAYERS (One Time Setup)
===============================================================================

1. Open the Player Gallery Seeder:
   
   cd C:\Users\nerdw\soccer_analysis
   python player_gallery_seeder.py

2. In the Seeder GUI:
   - Click "Load Video"
   - Navigate to a clear frame (use slider)
   - Draw a box around a player
   - Enter their name (e.g., "Kevin Hill")
   - Click "Add to Gallery"

3. Repeat for all key players

TIP: Select clear frames where the player's jersey is visible!

===============================================================================
  STEP 2: RUN ANALYSIS WITH AUTO-RECOGNITION
===============================================================================

1. Open the main GUI:
   
   python soccer_analysis_gui.py

2. Load any video and run analysis as normal

3. Watch the console for gallery matches:
   
   ‚úì Gallery match: Track #23 = Kevin Hill (similarity: 0.87)
   ‚úì Gallery match: Track #45 = John Doe (similarity: 0.82)

4. Check the output video - players will have their gallery names!
   
   Instead of "Player #23" ‚Üí "Kevin Hill #23"

===============================================================================
  TROUBLESHOOTING
===============================================================================

PROBLEM: "Player Gallery not available"
  ‚Üí Make sure player_gallery.py is in the soccer_analysis directory

PROBLEM: No gallery matches found
  ‚Üí Gallery might be empty - use player_gallery_seeder.py to add players
  ‚Üí Try lowering similarity threshold (edit combined_analysis_optimized.py line 3763)

PROBLEM: Wrong player identified
  ‚Üí Increase similarity threshold to 0.7+ for stricter matching
  ‚Üí Add more reference frames for the player

PROBLEM: "Re-ID tracker not available"
  ‚Üí Install PyTorch: pip install torch torchreid
